> A scaffolding tool for generating node modules.

## Usage

Use npm:

```shell
$ npm init @zce/nm <project>
```

Use yarn:

```shell
$ yarn create @zce/nm <project>
```

Use npx:

```shell
$ npx @zce/create-nm <project>
```

Globally install:

```shell
$ npm install @zce/create-nm -g
# or yarn
$ yarn global add @zce/create-nm
```

```shell
$ create-nm <project>
```
